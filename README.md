# 🎓 AcadBot – AI-Based Academic Support Chatbot
### TYBSC IT / BCA Semester 6 Project

---

## 📌 Project Overview
AcadBot is an intelligent academic support chatbot built using **Python (Flask)** and **NLP keyword-matching**. It answers student queries about syllabus, exams, attendance, fees, projects, library, and more — instantly and accurately.

---

## 🗂️ Project Structure
```
academic_chatbot/
├── app.py               ← Main Flask application (NLP engine + routes)
├── requirements.txt     ← Python dependencies
├── templates/
│   └── index.html       ← Chat UI (HTML + CSS + JS)
└── README.md
```

---

## ✅ How to Run

### Step 1 – Install Python (3.8+)
Download from https://python.org

### Step 2 – Install Flask
```bash
pip install flask
```
OR
```bash
pip install -r requirements.txt
```

### Step 3 – Run the App
```bash
cd academic_chatbot
python app.py
```

### Step 4 – Open in Browser
```
http://localhost:5000
```

## 📦 Deliverables Covered

- ✅ NLP-based intent classification (keyword scoring)
